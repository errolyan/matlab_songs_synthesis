Applications of (partial) morphing

This extension to morphing rate makes the morphing procedure a general purpose tool for speech manipulation. For example, a) duration control of each phonemic segment, and b) formant frequency control of each vowel segment, can be easily implemented using this extended procedure. The following sections illustrates how to do.

  • Duration control of segments
  • Formant frequency shift of each segment
Hideki Kawahara, November 7, 2005
